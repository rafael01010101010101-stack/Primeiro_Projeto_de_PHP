# 🐘 Laboratório de Lógica e Estruturas em PHP

Este projeto representa o meu primeiro contato prático com o desenvolvimento back-end utilizando **PHP**. O objetivo principal foi construir uma aplicação centralizada para consolidar fundamentos de lógica de programação, manipulação de variáveis, operadores matemáticos e processamento de dados dinâmicos no servidor.

A aplicação conta com uma interface de navegação simplificada que direciona o usuário para diferentes módulos funcionais de cálculo e conversão.

---

## 🛠️ Recursos e Funcionalidades

O sistema processa dinamicamente as seguintes operações baseadas nas entradas do usuário:

- **Operações Aritméticas Básicas:** Módulos individuais para Somar, Subtrair, Multiplicar e Dividir.
- **Cálculos Algébricos Avançados:** Implementação da **Equação de Segundo Grau** (Processamento de Delta e Bhaskara).
- **Módulo de Saúde:** Calculadora de **IMC** (Índice de Massa Corporal) com análise de dados.
- **Geometria Aplicada:** Funções para calcular a **área de um retângulo** e a **área de um triângulo**.
- **Lógica de Negócios (Financeiro):** Algoritmo para aplicar dinamicamente **18% de imposto** ao valor de um produto.
- **Conversão de Dados:** Script utilitário para **converter idade em dias**, trabalhando com manipulação de inteiros.

---

## 💻 Stack Tecnológica

- **PHP:** Responsável por toda a lógica de back-end, recebimento de requisições, processamento das fórmulas matemáticas e renderização dinâmica dos resultados.
- **HTML5 / CSS:** Estruturação e estilização da interface e dos botões para garantir uma navegação fluida e intuitiva.
- **Ambiente de Desenvolvimento:** Desenvolvido e testado utilizando servidores locais em ambiente VS Code.

---

## 📈 Aprendizados Consolidados

Desenvolver esta aplicação foi essencial para dominar os pilares iniciais do desenvolvimento back-end:
1. **Manipulação de Requisições:** Entendimento prático de como o servidor PHP captura dados enviados pelo front-end.
2. **Modularização e Lógica:** Aplicação prática de estruturas condicionais (`if/else`), operadores lógicos e tratamento de operações matemáticas complexas no código.
3. **Casamento Web Completo:** Comunicação entre a interface do usuário (Client-side) e o motor de processamento (Server-side).
